
# PublicPulse

PublicPulse is a Civic Complaint Management System developed using Spring Boot and MySQL.  
It helps citizens report issues like pollution, garbage, drainage, and road damage easily.

## Features

- User Login & Signup
- Complaint Submission
- Image Upload
- Live Location Detection
- Admin Dashboard
- Complaint Status Tracking

## Technologies Used

- Java
- Spring Boot
- MySQL
- Thymeleaf
- HTML/CSS/JavaScript

## Run Project

./mvnw spring-boot:run

Open:

http://localhost:8080
